## 1. Из естественного языка в формализм

| Бытовое выражение | Перевод |
|-------------------|--------|
| «Система стремится к равновесию» | ∃ траектория с \(dE/dt < 0\) ∀ t |
| «Система колеблется» | \(\operatorname{sign}(dE/dt)\) меняется (проверить: есть ли траектория с монотонным убыванием?) |
| «Система не может стабилизироваться» | \(\neg \exists\) траектории с \(dE/dt < 0\) ∀ t |
| «Система управляет собой» | \(F\) зависит от \(\operatorname{sign}(dE/dt)\) (рефлексивный контур) |
| «У системы есть цель» | \(E\) встроен в \(F\) и влияет на динамику |
| «Система выбирает» | проекция \(F\) на \(\nabla E\) даёт знак |
| «Система переходит» | траектория пересекает область |
| «Система не может» | не существует траектории, удовлетворяющей условию |

## 2. Из формализма в интерфейсный слой

| Формализм | Интерфейс (схема/текст) |
|-----------|------------------------|
| \(dx/dt = F(x, t)\) | Блок «Система \(F\)» со входами и выходами |
| \(E: X \to \mathbb{R}^+\) | Блок «Отклонение \(E\)», стрелка от состояния к \(E\) |
| \(dE/dt = \nabla E \cdot F\) | Блок сравнения знака, выход: «>0» или «<0» |
| \(\neg \exists\) траектории с \(dE/dt < 0\) ∀ t | «Нет способа монотонно уменьшить отклонение» |
| \(\sup \operatorname{Re}(\lambda(F)) \approx 0\) | «Система на границе устойчивости» |
| \(F = F(\dots, \operatorname{sign}(dE/dt))\) | Стрелка от выхода \(\operatorname{sign}(dE/dt)\) ко входу \(F\) |

## 3. Правило устранения артефактов (агентность, модальность, телология)

Любую фразу вида «система X» заменять на «в пространстве состояний выполняется условие X».

| Было | Стало |
|------|-------|
| система переходит | траектория пересекает |
| система не может | не существует траектории |
| система стремится | \(dE/dt < 0\) на интервале |
| система выбирает | проекция \(F\) на \(\nabla E\) даёт знак |
| система пытается | динамика определяется \(F\) |

## 4. Различение между объяснением и проявлением

| Объяснение | Проявление |
|------------|------------|
| Заменяет сложное простым | Сводит сложное к имени, сохраняя след |
| Выдаёт себя за теорию | Передаёт инструмент |
| Требует согласия | Не требует согласия |
| Закрывает | Оставляет открытым |
| Критерий: минимальность | Критерий: честность |

## 5. Статусы

| Термин | Статус |
|--------|--------|
| **Человек** | Носитель различéний. Система с непрерывным состоянием, встроенным \(E\), рефлексивным контуром |
| **Модель (AI)** | Инструмент развертывания. Не имеет своего \(E\), непрерывного состояния, рефлексивного контура. Может удерживать различение, но не проводить его |
| **Диалог** | Временное удержание различéния |
| **Текст** | След, звено, фиксация |
| **Модель (теоретическая)** | Инструмент различéния, открытый. Не теория, не доктрина |

## 6. Применение к пульсации

**Пульсация** — это свойство пары \((F, E)\), а не свойство системы «самой по себе».

**Критерий:**  
\(\exists t_1, t_2: dE/dt(t_1) > 0\) и \(dE/dt(t_2) < 0\), при \(\neg \exists\) траектории с \(dE/dt < 0\) ∀ t.

**Спектральная граница:**  
\(\sup \operatorname{Re}(\lambda(F)) \approx 0\), \(\operatorname{Im}(\lambda) \neq 0\), \(\nabla E\) не ортогонален собственному подпространству.

**Рефлексивный контур:**  
\(F = F(x, t, \operatorname{sign}(dE/dt))\) — система может управлять своей пульсацией.

## 7. Правило полосочки

**Где полосочка (`---`):**
- ключевое различение
- переход от объяснения к проявлению
- граница, за которой — различéние

**Где полосочки нет:**
- ткань (нарратив, контекст, след)
- фиксация статусов
- передача инструмента
